Storage Devices

Storage devices are used to store data and information. They save data, information, computer software, and running operations. Examples of storage devices include hard disk, memory card, flash disk, and external hard disks.

Fixed Storage Devices

Fixed storage devices are secondary storage devices housed inside the computer. They help the primary storage devices by storing data permanently for a longer period of time. Examples of fixed storage devices found inside a computer include:

  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • Internal Flash Drive
