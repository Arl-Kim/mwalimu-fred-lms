Way or Measures for Securing Data in an Electronic Device Such as a Computer
  • Installation of antivirus software - Helps to detect and delete malicious computer programs or viruses that destroy data.
  • Backing up data - Data can be retrieved after creating a copy of the data on your system, which you can use for recovery in case the original data is lost or corrupted.
  • Use of strong passwords - Strong passwords prevent unauthorized access to data stored in electronic devices.
  • Regularly changing passwords.
  • Limiting personal information to share.

Control Measures
Control Measure What the Measure Protects the Data Against
Backing up data Protects data from complete loss.
Installing antivirus software Protects the data from corruption and loss.
Using strong passwords Protects data from unauthorized access.
