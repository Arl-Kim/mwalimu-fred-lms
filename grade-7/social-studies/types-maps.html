Types of maps
    1. Topographical maps - This shows selected natural physical features on a small portion of a country.
    2. Atlas map -this is a collection of maps in one volume
    3. Sketch map -maps which are roughly drawn.
      4. A good sketch map should have the following characteristics:
        1. Neat and clear
        2. Title
        3. Frame
        4. Key
        5. Compass direction
