Recurring Decimals

These are decimal fractions in which a digit or a group of digits repeat continuously without ending.

  • ¹/₃ = 0.333333
  • ⁵/₁₁ = 0.454545454

We cannot write all the numbers, so we place a dot above a digit that is recurring. If more than one digit recurs in a pattern, we place a dot above the first and the last digit in the pattern.

E.g.
  • 0.3333……… is written as 0.3
  • 0.4545……… is written as 0.45
  • 0.324324……… is written as 0.324

Any division whose divisor has prime factors other than 2 or 5 forms a recurring decimal or non-terminating decimal.

Example

Express each as a fraction:

  1. 0.6
  2. 0.7˙3
  3. 0.˙15

Solution

a. Let r = 0.66666 - (i)

10r = 6.6666 - (ii)

Subtracting (i) from (ii): 9r = ⁶/₉ = ²/₃

b. Let r = 0.73333 - (i)

10r = 7.333333 - (ii)

100r = 73.33333 - (iii)

Subtracting (ii) from (iii): 90r = ⁶⁶/₉₀ = ¹¹/₁₅

c. Let r = 0.151515 - (i)

100r = 15.1515 - (ii)

99r = ¹⁵/₉₉ = ⁵/₃₃
